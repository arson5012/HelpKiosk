 ## 2. 블록도 설명
 <center>
     <img src="Readme_src/블록.png" alt="블록도">
 </center>

 1. 버튼 이벤트
     > 각 버튼 클릭 시 웹서버에 저장된 목적지, 시간, 좌석, 연령대별 가격 출력   

 2. 결제
     >  결제 확인 후 NFC Tag를 이용하여 결제
     >  결제 완료 정보를 DB서버에 저장 및 웹 서버에 반영
     
 3. 웹 서버
     >  목적지, 목적지별 가격 및 시간, 소요시간, 출발날짜, 좌석, 연령대별 가격 저장

 4. DB 서버
     > 미 결제 정보 및 결제 완료 정보 저장
